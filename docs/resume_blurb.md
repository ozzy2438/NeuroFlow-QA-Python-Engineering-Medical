# Resume and LinkedIn Blurb

## Resume bullet - concise

Built NeuroFlow QA, a 480MB+ synthetic brain MRI/fMRI quality-assurance pipeline in Python that generates NIfTI neuroimaging data, validates metadata, extracts QC metrics, flags scans for review, and produces reproducible reports.

## Resume bullet - stronger technical version

Developed NeuroFlow QA, a 100-subject synthetic brain MRI/fMRI pipeline using Python, NumPy, pandas, nibabel, and matplotlib. The project handles NIfTI neuroimaging files, extracts QC metrics including anatomical SNR, temporal SNR, motion, DVARS-style temporal variation, and brain-volume checks, then generates review flags, plots, validation notes, and an HTML report.

## LinkedIn / portfolio short pitch

NeuroFlow QA is a synthetic medical imaging data-engineering project focused on reliability rather than diagnosis. It shows how a healthcare AI pipeline can organize MRI/fMRI files, validate metadata, compute quality metrics, explain failure reasons, and produce evidence for human review.

## Interview explanation

I built this project to show that I understand the practical side of healthcare AI: data quality, reproducibility, traceability, documentation, and safety disclaimers. The project does not make clinical claims. It demonstrates how to prepare medical imaging data for downstream analysis in a controlled and inspectable way.
