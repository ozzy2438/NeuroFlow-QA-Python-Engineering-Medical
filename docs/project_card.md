# Project Card - NeuroFlow QA

## Short pitch
NeuroFlow QA is a synthetic brain MRI/fMRI quality assurance pipeline built for a medical imaging software portfolio.

## What it demonstrates
It demonstrates NIfTI file handling, synthetic medical image generation, metadata validation, quality-control feature extraction, rule-based scan flagging, and recruiter-friendly reporting.

## Important safety note
This project is not a diagnostic medical device. It uses synthetic data only and does not make clinical decisions.

## Current dataset
The project contains 100 synthetic subjects and is approximately 480MB+ on disk.

## Interview angle
The strongest message is: I understand not only modelling, but also medical imaging data quality, pipeline reliability, validation, documentation, and reproducibility.
